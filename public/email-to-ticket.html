Install the Gmail add-on so you can turn any email into a ticket from inside Gmail.

3 Install the Gmail add-on

One-time setup in Google Apps Script. Takes about 3 minutes.

  1. Open script.google.com and click New project.
  2. Replace the default Code.gs contents with the code below, then save (Ctrl/Cmd + S).
  3. Click the gear icon (Project Settings) → check Show "appsscript.json" manifest file.
  4. Back in the editor, click appsscript.json and paste in the manifest below. Save again.
  5. Click Deploy → Test deployments. Select Gmail, click Install, allow the requested permissions.
  6. Reload Gmail. Open any email. You'll see a new icon in the right-hand sidebar — click it.
  7. In the add-on sidebar, click Settings, paste your API token and App URL, save.
  8. Open an email, click the add-on icon, hit Create Ticket. Done.
